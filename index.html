2. Setup Commands

# Clone the repo
git clone https://github.com/hiren14/task_api_hiren_6354753667.git
cd task_api_hiren_6354753667

# Install dependencies
npm install

# Copy and edit environment variables
cp .env.example .env
# Set DB_HOST, DB_PORT, DB_USER, DB_PASS, DB_NAME, JWT_SECRET
# check is that database exists or not, if not then create it

# Run migrations
npm run migrate

# Start dev server
npm run dev

# API     → http://localhost:3000
# Swagger → http://localhost:3000/api/docs

3. API Endpoints (Swagger UI at /api-docs)

MethodEndpointDescriptionAuth
POST/auth/loginGet JWT tokenNo
GET/tasksList all tasks (paginated)Yes
GET/tasks/:idGet task by IDYes
POST/tasksCreate a taskYes
PATCH/tasks/:idUpdate a taskYes
DELETE/tasks/:idSoft delete a taskYes
GET/api-docsSwagger UINo

Note: Protected routes require Authorization: Bearer <token> header.
